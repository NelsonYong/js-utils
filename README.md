## How to use

- npm 

  ```sql
  npm install js-utils-pro -D
  ```

- pnpm 

  ```sql
  pnpm install js-utils-pro -D
  ```

- yarn 

  ```sql
  yarn add js-utils-pro -D
  ```

In code, you can

```typescript
import {isFns} from 'js-utils-pro'
isFns.isEmpty([]) //true
isFns.isNumber(222222) // true
//...
//Those methods are covered by test. They are safe and available.
```
